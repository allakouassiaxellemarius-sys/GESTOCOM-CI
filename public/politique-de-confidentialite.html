Alla Kouassi
Email : contact@gestocom.ci

2. Données collectées

GESTOCOM CI est une application de gestion commerciale fonctionnant principalement hors ligne. Voici les données traitées :

  • Données de gestion (ventes, stock, dépenses, fournisseurs) : stockées localement sur votre appareil via SQLite. Nous n'y avons pas accès.
  • Données de compte (nom, email, téléphone, mot de passe hashé) : stockées localement. En cas de connexion au cloud optionnel, synchronisées vers Firebase (Google).
  • Données de connexion : adresse IP, type de navigateur — collectées uniquement par l'hébergeur (Vercel) à des fins de sécurité.

3. Finalité du traitement

  • Fournir le service de gestion commerciale
  • Gérer votre compte utilisateur
  • Assurer la sécurité de l'application
  • Améliorer l'expérience utilisateur

4. Base légale

Le traitement est fondé sur votre consentement (utilisation de l'application) et sur l'exécution du service demandé.

5. Partage des données

Nous ne vendons ni ne partageons vos données personnelles avec des tiers, sauf :

  • Vercel (hébergement) — données de connexion techniques
  • Firebase / Google (synchro cloud optionnelle) — uniquement si vous activez la synchronisation cloud

6. Stockage et sécurité

  • Les données de gestion sont stockées localement sur votre appareil (SQLite chiffré)
  • Les mots de passe sont hashés avec bcrypt
  • La synchronisation cloud est chiffrée en transit (HTTPS/TLS)
  • Aucune donnée n'est transmise à des serveurs tiers sans votre action explicite

7. Vos droits

Conformément à la législation ivoirienne (Loi n°2013-450) et au RGPD, vous disposez de :

  • Droit d'accès : obtenir une copie de vos données
  • Droit de rectification : corriger vos données
  • Droit de suppression : supprimer vos données
  • Droit d'opposition : vous opposer au traitement
  • Droit à la portabilité : exporter vos données

8. Conservation des données

Vos données sont conservées tant que vous utilisez l'application. La suppression de votre compte entraîne la suppression définitive de toutes vos données.

9. Cookies

L'application utilise uniquement des cookies techniques nécessaires au fonctionnement (session, préférences). Aucun cookie publicitaire n'est utilisé.

10. Mineurs

L'application n'est pas destinée aux personnes de moins de 16 ans.
